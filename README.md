# BabyPainter
只适配了手机一加6的尺寸（2280*1080），6.28英寸全面屏，其他手机可能会有界面显示问题。
使用了ChooseButton，是一个圆盘菜单，可以从中选择颜色，具体请看https://github.com/maplefan/ChooseButton
